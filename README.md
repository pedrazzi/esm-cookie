# JavaScript Module Cookie

## Example for how to load the ES module in a browser:

```html
<script type="module" src="/assets/js/utils/js.cookie.js"></script>
```
<b>OR<b>
```html
<script type="module">
  import Cookies from '/assets/js/utils/cookie.js'

  Cookies.set('foo', 'bar')
</script>
```
